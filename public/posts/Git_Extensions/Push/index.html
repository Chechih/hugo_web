Git Extensions Push GitHub

七月 20, 2022

還沒安裝 Git Extensions 的可以參考這個

註冊

要在 GitHub 上 Push 當然也需要帳號拉~ 先到 GitHub 網站註冊，
還沒註冊的可以參考這個

GitHub 新增專案

再來就是點頭像，到專案去，按綠色的按紐新增

再來到新增畫面，填寫 Repository name ，專案名稱
再來按 Create repository 就完成 GitHub 的部分了，URL 的部分要先留下來等等用。

Git Extensions 安裝和 Commit

這部分可以參考這個

Push

如果 Git Extensions 和 Commit 好了，這時就可以 Push 了， 先到到想要 Git 的資料夾右鍵 Open_Repositories

然後按上箭頭

Url 填 剛剛在 GitHub 建立完專案得到的一組 Url 應該是長類似這樣 https:// github.com/XXX/OOO，
後面會再跳出 Remote repositories， 填個喜歡的 Name ，Url 跟剛剛一樣(https:// github.com/XXX/OOO)

途中可能會跳出 GitHub 登入

登入完畢後，應該就上去 GitHub 了 剛剛 GitHub Repositories 的頁面應該也會改變，這樣就上去了~
